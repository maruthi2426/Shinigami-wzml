========================================
HOW IT WORKS (COMPLETE FLOW)
========================================

USER COMMAND:
/leech "https://vegamovies.wedding/51369-anemone-2025-hindi-dual-audio-web-dl-720p-480p-1080p.html" -q "720p"

STEP 1: URL Detection
├─ Bot detects "vegamovies" in domain
├─ Calls direct_link_generator(url)
└─ Routed to webscrapper_handler()

STEP 2: WebScrapper Initialization
├─ Loads webscrapper.py module
├─ Initializes VegaMoviesScraper class
├─ Creates Selenium Chrome driver
└─ Logs: "[INFO] VegaMovies Scraper Initialized"

STEP 3: Page Scraping
├─ Loads VegaMovies page in browser
├─ Extracts title, season, episode info
├─ Identifies if MOVIE or SERIES
├─ Finds all quality options (480p, 720p, 1080p, 4K)
├─ Detects file sizes
└─ Logs: "[INFO] Found X download links"

STEP 4: Quality Filtering
├─ If -q "720p" provided, filters links
├─ Selects matching quality only
├─ Reduces processing time significantly
└─ Logs: "[INFO] Quality filter applied: 720p"

STEP 5: Shortener Resolution
├─ Visits Nexdrive shortener links
├─ Clicks "Verify" button
├─ Waits for page to load
├─ Extracts direct Google Drive links
├─ Handles multiple episodes in series
└─ Logs: "[SUCCESS] Resolved X direct links"

STEP 6: Return Direct Links
├─ Returns list of direct download URLs
├─ Each with metadata (title, quality, size, episode)
└─ Example: "https://drive.google.com/uc?id=xxx"

STEP 7: Auto Leech/Mirror Start
├─ Bot receives direct links
├─ Passes to aria2 download handler
├─ Starts leech/mirror/extract based on /cmd
├─ Files ready for upload automatically
└─ User receives progress updates
