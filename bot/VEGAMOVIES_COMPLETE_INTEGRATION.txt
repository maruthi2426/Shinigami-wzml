HOW IT WORKS - COMPLETE FLOW:
=============================

USER INPUT:
-----------
/leech https://vegamovies.wedding/movie -q 720p

PROCESSING:
-----------
1. Bot receives command
2. arg_parser extracts URL and quality filter (-q 720p)
3. URL validation check
4. direct_link_generator() is called with URL and quality_filter
5. Domain detection identifies vegamovies.wedding
6. webscrapper_handler() is triggered
7. webscrapper.py loads and initializes VegaMoviesScraper
8. Selenium driver fetches page with JavaScript rendering
9. BeautifulSoup parses HTML
10. Downloads links extracted with quality metadata
11. Quality filter applied (matching "720p")
12. Shortener links resolved to direct URLs
13. Direct Google Drive/Pixeldrain links returned
14. aria2 begins download automatically

DOWNLOAD STARTS:
----------------
✓ Quality: 720p
✓ Direct link obtained from scraper
✓ aria2 downloads file
✓ Upload/Mirror process begins automatically
