(Quality matching is case-insensitive)


HOW IT WORKS IN REAL-TIME:
===========================

1. User sends command with URL and quality
2. Bot detects VegaMovies domain
3. Selenium opens headless browser
4. Page fully loads with JavaScript
5. BeautifulSoup parses all download links
6. Links filtered by quality parameter
7. Shortener links resolved to direct URLs
8. First direct link returned
9. aria2 starts download immediately
10. Process shown in real-time logs
